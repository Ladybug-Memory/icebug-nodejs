# Nodejs bindings for icebug

[Icebug](https://github.com/Ladybug-Memory/icebug) is a zero-copy Apache Arrow based graph analytics package.
It contains more than 100 graph algorithms.

This is a nodejs wrapper for the library.

## Installation

Copy pre-built icebug to vendor/{include,lib}. This will be automated via CI in the future.

```
npm run test
```
